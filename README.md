## Iniciando o projeto

Instale o projeto com o comando:

```bash
npm install
```

Execute o servidor de desenvolvimento:

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000) com seu navegador para ver o resultado.
